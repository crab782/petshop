## 3. 业务场景覆盖

### 3.1 完整业务流程
1. **用户注册登录** → **浏览商家** → **预约服务** → **服务完成** → **评价服务**
2. **用户注册登录** → **浏览商品** → **添加到购物车** → **下单支付** → **确认收货** → **评价商品**
3. **商家注册** → **平台审核** → **发布服务** → **接收预约** → **提供服务** → **处理订单**
4. **平台管理员** → **审核商家** → **管理用户** → **处理订单** → **管理评价**

### 3.2 边界情况
1. **商家状态**：包含approved, pending, rejected三种状态
2. **订单状态**：包含pending, confirmed, completed, cancelled四种状态
3. **评价状态**：包含approved, pending两种状态
4. **商品库存**：包含正常库存和低库存情况
5. **服务价格**：包含不同价格区间的服务
