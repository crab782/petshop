# 登录后路由跳转和403问题分析 - 验证检查清单

- [ ] 检查登录成功后是否跳转到用户首页（/user/home）
- [ ] 检查是否不再跳转到商家登录页（/merchant/login）
- [ ] 检查用户端接口是否能正常访问（返回200 OK）
- [ ] 检查商家登录页是否只对商家开放
- [ ] 检查其他用户端功能是否正常运行
- [ ] 检查登录状态是否正确存储和管理
- [ ] 检查API调用是否正确传递认证信息
- [ ] 检查后端权限控制是否正确验证用户身份和权限
- [ ] 检查路由守卫是否正确处理不同角色的路由权限
- [ ] 检查整个登录流程是否流畅，无异常跳转
